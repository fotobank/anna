Перед вами скрипт гостевой книги с возможностью премодерации.

Скрипту для нормальной работы требуется следующее:
- интерпретатор языка PHP (тестировалось на версии 4.2.0)
- база данных MySQL (тестировалось на версии 3.23.53)
- web-сервер, на котором всё это хозяйство работает (тестировалось на Apache версии 1.3.22)

Итак, чтобы сделать этот не очень сложный скрипт рабочим, надо выполнить нижеследующее:

1. Открыть файлик config.inc.php и внести туда свои значения для SQL-хоста, для логина и пароля к этому хосту, и для базы данных, которая будет использоваться в работе;

2. Установить соединение со своим SQL-хостом и залить в него инструкции из файла sql.inc. Он создаст в выбранной вами базе данных три таблицы (gb_posts, gb_replies, gb_users). Излишне говорить, что база, в которой вы это создаёте, должна быть той же, что и указанная в файле config.inc.php :)

3. Упомянутый выше файлик помимо пустых таблиц создаст одну учётную запись с логином и паролем - 'admin'. Настоятельно рекомендуется если не логин, то хотя бы пароль изменить. Для этого уже можно воспользоваться данным скриптом - запускайте файлик auth.php, авторизуйтесь (admin:admin) и идите в раздел управления пользователями: там можно(нужно!) будет изменить пароль, логин и ввести адрес электронной почты, на который будут высылаться новые сообщения;

4. Вот и всё. Установка завершена и гостевая книга готова к работе.

PS. для изменения дизайна гостевой редактируйте шаблоны из папки _templates. Для работы с шаблонами используется
 класс "Fast Template" (http://www.thewebmasters.net/).
